When not to use this component

Do not use the GOV.UK footer on services not hosted on GOV.UK domains. Avoid mixing GOV.UK and PINS branding on the same page.

How it works

GOV.UK footer

{% from "govuk/components/footer/macro.njk" import govukFooter %} {{ govukFooter({}) }}

  • Mandatory for GOV.UK-hosted services.
  • Light blue background, thick blue top border.
  • Includes crown logo and “© Crown copyright”.
  • Optional help links (must be consistent across pages).
  • Font: GDS Transport.

PINS footer

  • Link 1
  • Link 2
© Planning Inspectorate

  • Must not include GOV.UK branding or OGL text.
  • Custom content relevant to Planning Inspectorate.
  • Feedback links should point to PINS-specific pages.
  • Licensing must reflect non-GOV.UK status.
  • Font: Arial or Sans Serif.

Accessiblity

Footer links must be clearly labelled and consistently placed.

Colour contrast must meet WCAG standards.
