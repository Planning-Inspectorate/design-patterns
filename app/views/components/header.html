When not to use this component

Do not use the GOV.UK header on services not hosted on GOV.UK domains. Avoid mixing GOV.UK and PINS branding on the same page.

How it works

GOV.UK header


{% from "govuk/components/header/macro.njk" import govukHeader %} {{ govukHeader({ homepageUrl: "#" }) }}

  • Mandatory for GOV.UK-hosted services.
  • Includes GOV.UK crown and logotype.
  • Blue background, height 60px.
  • Service name and navigation must be handled separately using the Service navigation component.
  • Font: GDS Transport.

PINS header





  • Must not include GOV.UK branding.
  • Visually distinct from GOV.UK header.
  • Colour palette should align with MHCLG colourways but use PINS assets.
  • Licensing must reflect non-GOV.UK status.
  • Font: Helvetica Neue, Arial or Sans Serif.
  • Implementation details will be added once a shared approach is agreed.

Accessiblity

All headers must meet WCAG 2.2 standards.

Link colours must pass contrast checks (e.g. Teal #10847E for AAA compliance in Back Office).
